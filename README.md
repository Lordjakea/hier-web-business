# Hier Business Web Rebuild

This is a fresh UI-first rebuild for the business dashboard in a Trello-inspired style using the Hier brand colours.

## Included
- Login screen
- Candidate management board
- Shared dashboard shell with header, sidebar, app launcher, and account menu
- Placeholder pages for Analytics Pro, Job Posts, Promote, Billing, Profile Management, and Policies
- Tailwind setup and reusable components

## Run locally
```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Notes
- This build is intentionally mocked for now so you can approve layout and proportions first.
- The dashboard is ready for backend wiring after the visual pass.
- Replace the mock data in `lib/mock-data.ts` with real API data once you are happy with the design direction.
